RoundsBoard finally has the long-awaited pairing module, which in the future will completely replace manually uploading files from external pairing programs.

The current version is still in **beta** and offers fairly limited functionality:

- supports one tournament system: *McMahon*,
- the list of tiebreakers is limited (but sufficient to run a "standard" tournament),
- supports one player seeding method: *Split & Slip*,
- does not support handicap tournaments, complex supergroups, or more advanced configurations.

Even so, for simple tournaments without "unusual" configurations, you can already use it.

In future versions the module will be expanded to offer at least the same functionality as external pairing programs.

The pairing module is located in the *Panel*, in a given tournament's configuration, under the *Pairing* tab.
